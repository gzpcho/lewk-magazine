# Lewk Magazine

## Usage

1. Clone the repository
2. Create and activate a virtual environment
3. Install dependencies

```bas
# For API
pip install -r requirements.txt

# For client
cd client
npm install
```

3. Set environment variables
4. Create a build - `npm run build`
5. Serve the app - `flask run`
